Class: ParseResult

ParseResult

new ParseResult(value, next, start, matched)

A ParseResult holds the result of a successful parse operation. It contains the value which was parsed, a Stream pointing to the remaining unconsumed input, and some useful metadata.

Parameters:
Name Type Description
value any

The value which this parse operation produced.
next Stream

The position if the remainder of the input stream.
start Stream

The position at which this parser began parsing.
matched string

The exact string this parser consumed.
